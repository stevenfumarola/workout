# Workout Manager — GitHub Pages Setup

## Struttura
```
workout/               ← nome del repository
  index.html           ← Workout Manager (solo tuo)
  alessandra/          ← Scheda Alessandra Casillo
  francesco/           ← Scheda Francesco Casillo
  simone/              ← Scheda Simone Polizzi
```

## Setup (una volta sola, ~5 minuti)

1. Vai su github.com → New repository
2. Nome repo: `workout`
3. Visibilità: **Public** (necessario per GitHub Pages gratuito)
4. Carica tutti i file di questa cartella nel repo
5. Settings → Pages → Source: **Deploy from branch** → branch: `main` → folder: `/ (root)`
6. Dopo ~2 minuti il sito è live

Sostituisci `TUO-USERNAME` con il tuo username GitHub.

## Aggiungere un nuovo paziente

1. Apri il Workout Manager → carica la scheda xlsx del paziente
2. Premi 🌐 Pubblica → scarica la cartella del nuovo paziente
3. Carica la cartella nel repo GitHub → link attivo in 2 minuti
